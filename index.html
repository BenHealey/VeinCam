About

VeinCam is an educational tool for vein visualisation. It uses near infrared (nIR) light and a nIR-sensitive camera to stream a real-time vein-enhanced video image. In addition to a custom PCB to control the nIR light source, the device employs widely available low-cost components from the Raspberry Pi educational foundation.

Much of VeinCam's development occurred during ANU Engineering capstone projects in 2018 and 2019. Created as an open-source project, anyone who may find VeinCam useful for learning can find the information necessary to make one in the associated GitHub repository .

Note that VeinCam is not a certified medical device and is therefore not intended for use in a clinical setting.

Repository structure

The VeinCam GitHub repository contains schematics and code for both the hardware and software components of the device. The folders for each component contain 'readme' files outlining their contents and key aspects of functionality.
